# Codex Project Memory

This directory stores persistent project context for future Codex CLI sessions.

## Files

- `assessment.md`: architecture, current project assessment, and milestone view
- `session-state.md`: current branch state, working assumptions, and resumable
  session notes
- `next-steps.md`: deferred work and candidate follow-up directions

## Conventions

- Keep entries concise and factual.
- Prefer stable facts over speculative notes.
- Update these files whenever a milestone materially changes the project.
